## Avant tout
1. Cloner
2. Ouvrir un terminal dans le dossier `pfc/`.
3. Lancer `npm install`.

## Lancer
1. Compiler le client : `npm run build`.
2. Lancer le serveur : `npm start`.
3. aller dans localhost:8080

## Ce que j'ai fait :
- `server/` : Gestion des pages & des évènement serveur
- `client/` : Code client , css : gestion des actions
- `public/` : Pages html & images
